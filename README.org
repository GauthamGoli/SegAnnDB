##Testing -

For starting the test suite - 

Download the test profile from -
https://drive.google.com/open?id=0BxbS0oJuMJTHS1RGeGtvWmp3VzA

install selenium using - 
  `pip install selenium=2.53.2`

Commands -
  `cd SegAnnDB`
  
  `cd tests`

edit line number 19 to correspond to the current path

  `python tests.py`
